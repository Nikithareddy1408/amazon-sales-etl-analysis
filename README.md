# Amazon Sales ETL Analysis

## Overview
This project analyzes Amazon sales data to identify revenue trends, high-performing regions, and profitable product categories.

## What I did
- Performed ETL to clean and structure raw data
- Built data transformations using Python (Pandas)
- Analyzed trends across regions and product categories
- Identified high-revenue regions and top-performing products

## Tech Stack
Python, Pandas, NumPy, SQL, Jupyter Notebook

## Key Results
- Processed 500K+ records
- Analyzed $137M+ in total revenue
- Identified high-profit product categories and regions
